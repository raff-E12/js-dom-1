# Progetto Js-DOM-1

## Panoramica del Progetto
Questo progetto dimostra l'uso di JavaScript per manipolare il DOM (Document Object Model) per cambiare lo stile e la funzione di un elemento. Il progetto coinvolge specificamente uno switch che cambia la classe di un elemento "lampadina" e cambia anche lo stile dell'elemento body.

## Funzionalit√†
- Cambia la classe di un elemento lampadina tra "off" e "on".
- Cambia la classe del body tra "light-mode" e "dark-mode".
